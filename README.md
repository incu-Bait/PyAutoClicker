## **What To Expect**

<div align="center">

| Feature | Description |
|---------|-------------|
|  **Really Fast Clicking** | Like REALLY FAST **BEWARE !!**|
|  **Multiple Themes** | Dark, Light & Neon themes |
|  **Custom Hotkeys** | Fully rebindable keyboard shortcuts |
